# The POST 

@argotactical

Any conversation with argo will include a meeting POST:
- Purpose - we will always define the purpose for the conversation
- Outcome - we will always state what the outcome is 
- Structure - we will clearly articulate who is talking about each topic, and an approximation of time needed
- Time - we will tally up the total time needed for the diuscussion. 

We use the POST to be respectful of everyones time and to allow for preparation
